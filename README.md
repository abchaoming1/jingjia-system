# 超全竞价 - 无敌

这是一个面向日常实战复盘使用的集合竞价学习与案例网页。

站点结构分为两部分：

1. 第一部分：基础知识与基本要点
2. 第二部分：案例教学库

当前版本重点特性：

- 首页工作台化，适合每天打开直接复盘
- 第二部分支持 `清单模式 / PPT 模式` 双视图切换
- 支持关键词、题材、形态、清单标签、批注状态组合筛选
- 支持本地批注、已复盘标记、自定义标签
- 对手机端做了目录抽屉、底部快捷栏和输入区优化

## 本地预览

在当前目录执行：

```bash
python -m http.server 8765
```

然后打开 `http://127.0.0.1:8765/`
